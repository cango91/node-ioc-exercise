# Task Manager

This is an exercise in Clean Architecture and IoC/DI pattern in Node ecosystem.

## Core Features
1. **User Registratio/Login**: Users can login and register
2. **Task Management**: Users can create, update, delete and list their tasks

## Getting Started
Clone repository.
`git clone https://github.com/cango91/node-ioc-exercise.git`

Install server dependencies with npm
`npm i`

Install client dependencies
```
cd client
npm i
```

Run client with `npm run serve` or build it with `npm run build`


Run server with `npm run build` on root directory
